Straight Skeletons

Given walls of all the same height, how can we construct a polygonal roof such that all faces have the same slope?

The straight skeleton is computed by a shrinking process in which edges move inwards in a direction normal to itself and vertices move along angular bisectors. The shrinking continues until:

1) The polygon's area becomes zero

2) An edge event occurs when an edge length decreases to 0 or

3) A split event occurs when a reflex vertex collides into an edge.
